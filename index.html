How to Add This Signature in Zoho Mail:

  1. Login to Zoho Mail
  2. Click the Gear icon (⚙) in the top-right corner
  3. Go to Mail Settings → Signatures
  4. Click + Add to create a new signature
  5. Select Rich Text Editor mode
  6. Click the </> icon (HTML source)
  7. Paste your HTML email signature code
  8. Click Save and assign the signature to the desired “From” address
  9. Send a test email to verify formatting
